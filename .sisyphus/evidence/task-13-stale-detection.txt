================================================================================

The cleanup script uses this algorithm:

2. Look for timestamp fields in order of priority:
   a. Direct: timestamp, time, ts, datetime
   b. Nested: last_result.timestamp
   c. Nested: inflight.timestamp
3. Parse ISO 8601 timestamp
4. Calculate age: now_utc - timestamp_utc
5. If age > STALE_THRESHOLD_HOURS (default 24):
   - Mark as stale
   - If NOT in allowlist -> recommend clear
   - If in allowlist -> preserve (log warning)
