### Trigger condition (lines 648–670)
When `table_name in ("IsNewSet", "IsNewFW", "IsNewWeather")` AND `self._cs.pending_frame is not None`:
  1. `setting_frame = self._cs.pending_frame`
  2. `self._cs.pending_frame = None`                          ← frame consumed
  3. `self._cs.pending["sent_at"] = time.monotonic()`         ← delivery timestamp
  4. `self._cs.pending["delivered_conn_id"] = conn_id`        ← ✅ OWNER STAMPED
  5. Frame written to `box_writer`
  6. `stats["acks_local"] += 1`

### Key finding
`delivered_conn_id` is stamped on `self._cs.pending` immediately before the setting
frame is written to the BOX connection, and only if `conn_id is not None`. This
ensures `maybe_handle_ack` will correctly attribute the ACK/NACK response to
exactly the same connection that delivered the setting.

### Scoped-state contract compliance
- `send_to_box`: initialises `pending` dict WITHOUT `delivered_conn_id` (not yet delivered)
- `_process_frame_offline`: stamps `delivered_conn_id = conn_id` at delivery time ✅
- `maybe_handle_ack`: validates `conn_id == pending["delivered_conn_id"]` before processing ✅
