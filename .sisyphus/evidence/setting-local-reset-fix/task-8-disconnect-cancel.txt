- The timeout tasks are properly cancelled, preventing stale callbacks

#### Code Changes:
1. `control_pipeline.py` - note_box_disconnect() now cancels timeout tasks:
   ```python
   async def note_box_disconnect(self) -> None:
       async with self.lock:
           tx = self.inflight
           if tx is None:
               return
           if tx.get("stage") in ("sent_to_box", "accepted"):
               tx["disconnected"] = True
           for task in (self.ack_task, self.applied_task):
               if task and not task.done():
                   task.cancel()
           self.ack_task = None
           self.applied_task = None
   ```

2. `control_settings.py` - clear_pending_on_disconnect() clears pending state:
   ```python
   def clear_pending_on_disconnect(self) -> None:
       self.pending = None
       self.pending_frame = None
   ```

3. `proxy.py` - handle_connection finally block calls cleanup:
   ```python
   if self._cs:
       self._cs.clear_pending_on_disconnect()
   ```
