### 3. Safe Access Pattern in Activation

```python
# proxy.py:_activate_session_twin_mode_if_needed()
sidecar_adapter = getattr(self, "_sidecar_adapter", None)
if sidecar_adapter is not None:
    if not sidecar_adapter.orchestrator.is_active():
        sidecar_adapter.record_activation()
    if not await sidecar_adapter.check_and_deactivate():
        return
```

**Error handling:**
- `getattr(self, "_sidecar_adapter", None)` returns None if attribute doesn't exist
- `if sidecar_adapter is not None:` check prevents AttributeError
- Falls through to legacy activation path if orchestrator unavailable
