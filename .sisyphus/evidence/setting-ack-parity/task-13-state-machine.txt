FIRST DIVERGENCE
----------------
Transition index: 2
Cloud: DELIVERED → ACKED (success, via tbl_events observation)
Mock:  DELIVERED → CONN_CLOSED (BOX closes connection after 24s without ACK)

Root cause: In cloud mode, proxy is passive (transparent forwarding). In mock mode, proxy is 
the ACK endpoint. BOX closes connection before mock proxy receives ACK, and timeout fires on 
a different connection (cross-session timeout).
